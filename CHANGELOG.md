# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-10

### Added

-   Initial release of react-native-package-checker
-   Native Android module for detecting installed packages
-   `isPackageAvailable()` method to check if a specific package is installed
-   `checkPackagesByPattern()` method to find packages matching a regex pattern
-   Full TypeScript support with type definitions
-   Expo SDK 52 config plugin support
-   React Native 0.73+ compatibility
-   Automatic Android manifest permission handling via Expo plugin
-   Comprehensive documentation and usage examples
